E-COMMERCE PRODUCT CATALOG
A modern, fully-functional e-commerce product catalog built with React, featuring advanced filtering, cart management, and a polished user interface using ShadCN UI components.

🎯 FEATURES
Product Browsing

Product Grid Display - Responsive grid layout with product cards
Category Filtering - Filter products by categories (Beauty, Fragrances, Furniture, Groceries, etc.)
Search Functionality - Real-time debounced search by product title
Price Sorting - Sort products by price (Low to High, High to Low) and rating
Pagination - Server-side pagination for efficient data loading

Product Details

Detailed Product Pages - View complete product information
Image Carousel - Browse through multiple product images
Customer Reviews - Read product reviews with ratings
Stock Information - Real-time stock availability display

Shopping Cart

Add to Cart - Add products from listing or detail pages
Cart Management - Update quantities, remove items
Real-time Updates - Live cart count badge in navbar
Persistent Cart - Cart data saved to localStorage
Price Calculation - Automatic total price calculation

User Experience

Responsive Design - Optimized for mobile, tablet, and desktop
Loading States - Skeleton loaders for better UX
Error Handling - User-friendly error messages
Toast Notifications - Feedback for cart actions
Empty States - Clear messages when no products found


🛠️ TECH STACK
Core

React 18 - UI library
Vite - Build tool and dev server
React Router v6 - Client-side routing

State Management

useReducer - Complex state management (products, cart)
Context API - Global cart state
Custom Hooks - Reusable logic (useProducts, useCart, useDebounce)

UI & Styling

ShadCN UI - Accessible component library
Tailwind CSS - Utility-first styling
Lucide React - Icon library

API Integration

DummyJSON API - Product data (https://dummyjson.com)
Async/Await - Modern API handling
Error Boundaries - Production-safe error handling
